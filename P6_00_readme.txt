(10) notes.csv >> fichier csv contenant les billets d'entraînement


2. Exécution
Dans R studio :
Pour exécuter le code R du projet, ouvrir le fichier P6_01_code.rmd et exécuter l'ensemble des blocs
Pour exécuter le code R du détecteur :
	a. placer le fichier contenant les billets à tester à la racine du dossier (par exemple C:\Users\maxim			    \Desktop\Formation Data Analyst\Projet 6\P6_vayne_maxime)
	b. le fichier à tester doit être au format .csv, être nommé "example.csv" (prendre le fichier example.csv 	    comme modèle)
	b. ouvrir le fichier P6_03_detecteur.rmd et exécuter l'ensemble des blocs

En cas de problème de lecture ou d'ouverture des fichiers csv contenant les données, changer le "working directory" à la racine du dossier, par exemple : "par exemple C:\Users\maxim\Desktop\Formation Data Analyst\Projet 6\P6_vayne_maxime".

Pour visualiser l'analyse & le détecteur sans exécuter, ouvrir les fichiers P6_02_code_html et P6_04_detecteur_html dans un navigateur internet.
